http
https
2.Domain: Domain make it easy for users to access websites without needing to remember IP addresses.There are two types in Domain
Top Level Domain
3.Path: The Path specifies the location of the resource on the server.

Agile Methodology


Agile Methodology is one most used method in developing of a project. It uses Sprint Method in division work to the developers like sprint1,sprint2..etc.for every sprint they will provide a time period.After every sprint the code is sent to the testing team,if testing team accept the code then it moves for sprint 2,else it sent to sprint1.There is a Scrum Call during the project,they will track the sprint details,clarify the developer queries and gives some instructions.The Scrum call is Managed by Scrum Master.The specialisation of this method is testing is done after every sprint unlike other methods that improves efficiency of the system.

Waterfall Methodology

It is one of the project implementation method mostly opted for Smaller Projects.In this method the testing is done after the completion whole development.It is mostly used method followed by agile.

SDLC(Software development Life Cycle):



Software Development Life cycle is the process for Developing project.It consist several steps

  • Requirement Analysis:Identify and document the requirements.clients define project goals,scope.

  • Planning:Plan for resources and analyse the budget.

  • Designing:Create architecture and Design features of the system.

  • Implementation:Developers write the code according to the design specifications.

  • Testing:The software is tested to find bugs and also the user acceptance.

  • Deployment:The system is made available for users.

  • Maintainance:The system is monitored after the Deployment.

    • Devops


  • The Devops team will secure data Deployment through CI/CD Pipelines.
  • They will adjust the storage Requirement by upscaling and downscaling.
  • They ensure Maintainance of the project after development.
