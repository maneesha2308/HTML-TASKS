VISUAL STUDIO CODE

Visual studio code (VS code) is a lightweight but powerful source code editor developed bt Microsoft,It supports a wide range of programming languages and comes with features.

Features of VS code

  • Syntax Highlighting and Auto-Completion for most languages.

  • Integrated Terminal for running scripts or commands

  • Extensions Marketplace to add functionality,such as languages specific support,linters,debuggers,themes and more.

  • Customoizable Interface with user-defined settings and shortcuts.

  • Built-in Debugging for a seamless coding experience.
